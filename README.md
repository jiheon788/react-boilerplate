## Usage
### `router`
```javascript
// src/meta/routerMeta.ts
const routerMeta: RouterMetaType = {
  ${Component name}: ${path},
};
```

## Getting Started

#### Install
```
npm i
```
#### Build
```
npm run build
```
#### Start
```
npm start
```
#### env

- Fill out .env for your api key & server url
```
REACT_APP_SERVER_URL = ${your server url}
```
